Website Evaluations

Example Good Site (Apple)

Example Good Site is a beautifully designed website that offers a seamless user experience.

  • Contrast: The site uses contrasting colors effectively to highlight important content and improve readability.
  • Repetition: Consistent design elements are used throughout the site, creating a cohesive look and feel.
  • Alignment: Elements on the site are well-aligned, making it easy for users to follow the content and navigate the site.
  • Proximity: Related content is grouped together, making it easy for users to find relevant information.

Example Bad Site (Yvette)

Example Bad Site is a poorly designed website with cluttered layout and confusing navigation.

  • Contrast: The site lacks contrast, making it difficult to read text and identify important content.
  • Repetition: Inconsistent design elements create a disjointed appearance and confuse users.
  • Alignment: Elements on the site are poorly aligned, making it difficult to follow the content and navigate the site.
  • Proximity: Content is randomly placed without clear grouping, making it challenging for users to find relevant information.
